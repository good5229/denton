# 시군구 합계와 시도 분기 총량 정합성 진단

## 목적

연간 시군구 GRVA를 분기 배분하면 각 시군구의 연간 합계는 공식 벤치마크와 100% 일치한다. 이는 예측 성능이라기보다 제약조건의 결과다. 더 의미 있는 검증은 하위 시군구 분기 추정치를 합산했을 때 상위 시도 분기 총량과 얼마나 가까운지 보는 것이다.

이번 진단은 서울시뿐 아니라 모든 시도와 산업에 대해 `시군구 합계 - 부모 시도 분기 GVA`를 계산한다.

## 산출물

모든 CSV는 CP949로 저장했다.

| 파일 | 설명 |
| --- | --- |
| `sigungu_parent_quarterly_consistency.csv` | 시도 × 산업 × 분기별 하위 시군구 합계와 부모 시도 총량 비교 |
| `sigungu_parent_consistency_by_area.csv` | 시도별 요약 |
| `sigungu_parent_consistency_by_sector.csv` | 산업별 요약 |
| `sigungu_parent_consistency_by_area_sector.csv` | 시도 × 산업별 요약 |
| `sigungu_parent_consistency_by_year.csv` | 연도별 요약 |
| `sigungu_parent_consistency_top_errors.csv` | 절대오차 상위 100건 |
| `sigungu_parent_consistency_missing_child.csv` | 부모 시도 총량은 있으나 하위 시군구 산출물이 없는 조합 |
| `sigungu_parent_consistency_summary.csv` | 전체 요약 |

## 현재 결과

비교 가능한 `시도 × 산업 × 분기` 조합은 4,260건이다. 부모 시도 총량은 있으나 하위 시군구 산출물이 없는 조합은 1,180건으로 별도 분리했다. 결측 조합을 `-100%` 오차로 계산하면 진단이 왜곡되므로, 정합성 지표는 비교 가능한 조합만 대상으로 계산했다.

| 지표 | 값 |
| --- | ---: |
| 비교 가능 행 | 4,260 |
| 하위 시군구 산출물 결측 행 | 1,180 |
| 전체 MAPE | 0.381704% |
| 전체 WMAPE | 0.106194% |
| 전체 aggregate percent error | 0.011735% |

산업별로는 `D00` 전기·가스의 정합성 차이가 가장 크다.

| 산업 | 비교 행 | MAPE | WMAPE | aggregate percent error |
| --- | ---: | ---: | ---: | ---: |
| 전기·가스 | 268 | 3.726226% | 2.427932% | -1.242293% |
| 광업 | 260 | 1.069389% | 0.228667% | 0.016800% |
| 운수 및 창고업 | 268 | 0.424701% | 0.572203% | 0.148956% |
| 제조업 | 268 | 0.097235% | 0.095011% | 0.079478% |

절대오차 상위 행은 주로 `경기도 × 전기·가스 × 2023년`, `전북특별자치도 × 전기·가스 × 2023년`에 집중된다. 따라서 전기·가스 산업은 기존에 별도로 검토한 외생변수 후보뿐 아니라, 시군구별 발전·전력판매·가스공급 관련 보조자료를 추가로 찾을 필요가 있다.

## 해석

- `mape`: 각 분기별 percent consistency error의 절대값 평균
- `wmape`: 절대오차 합계를 부모 시도 총량 합계로 나눈 값
- `aggregate_percent_error`: 전체 합계 기준의 방향성 있는 오차율

이 값은 “예측값과 실제값의 오차”라기보다, 서로 다른 공표 계열을 사용했을 때 하위 합계와 상위 총량이 얼마나 정합적인지를 보여준다.

## 왜 완전 일치하지 않는가

시군구 연간 GRVA와 시도 GRVA는 공표 단위와 연쇄가격 처리 방식이 다를 수 있다. 특히 실질 연쇄가격 계열은 하위 지역이나 산업을 단순 합산했을 때 상위 총량과 완전히 일치하지 않을 수 있다. 따라서 이 진단은 `0%`가 반드시 나와야 하는 회계 검증이 아니라, 모델의 공간 합산 안정성을 보는 품질 지표다.

## 활용

오차가 큰 `시도 × 산업` 조합은 다음 개선 후보가 된다.

1. 부모 시도 분기 지표 대신 해당 시군구 또는 인접권역의 보조 지표를 찾는다.
2. 시군구 연간 벤치마크와 시도 분기 총량을 동시에 만족시키는 2방향 벤치마킹을 적용한다.
3. 연쇄가격 합산 불일치가 큰 산업은 명목값 또는 디플레이터 재조정 방식도 별도로 검토한다.
4. 부모 총량은 있으나 하위 시군구 산출물이 없는 조합은 예측 오차가 아니라 coverage gap으로 관리한다.
