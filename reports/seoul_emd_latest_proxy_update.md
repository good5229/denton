# 서울 읍면동 최신 프록시 가중치 갱신

## 목적

기존 읍면동 GVA 프록시 배분은 KOSIS 2015년 경제총조사의 읍면동×산업대분류 매출액·종사자수·사업체수 가중치를 사용했다. 이 방식은 전국 공통 적용에는 유리하지만, 서울처럼 행정동 경제구조가 빠르게 변하는 지역에서는 가중치 노후화 문제가 크다.

이번 단계에서는 서울시 `한눈에 보는 사업체` 통계지도 API의 2024년 행정동 사업체/종사자 자료를 사용해 서울 읍면동 배분 가중치를 갱신하고, 2015년 경제총조사 기반 배분과 비교했다.

## 구현

추가 스크립트:

```text
scripts/update_seoul_emd_proxy_weights.py
```

생성 파일:

| 파일 | 설명 |
|---|---|
| `seoul_emd_business_proxy_2024.csv` | 서울시 2024 행정동 사업체/종사자 원천 프록시 |
| `seoul_emd_business_weights_2024.csv` | 자치구 내부 행정동별 2024 최신 가중치 |
| `seoul_emd_quarterly_gva_estimates_2024_proxy.csv` | 2024 최신 가중치로 재배분한 서울 읍면동 분기 GVA |
| `seoul_emd_2015_vs_2024_proxy_comparison.csv` | 산업별 2015 가중치 vs 2024 가중치 비교 |
| `seoul_emd_2015_vs_2024_proxy_summary.csv` | 행정동 총량 기준 비교 요약 |
| `seoul_emd_2024_proxy_allocation_skipped.csv` | 재배분 스킵 행 |

## 해석

2015년 고정 가중치와 2024년 종사자수 기반 가중치의 차이가 상당히 크다. 이는 읍면동 프록시 추정에서 최신 공간 가중치 갱신이 필수라는 뜻이다.

다만 이번 최신 가중치는 산업대분류별 매출액이 아니라 2024년 행정동 전체 종사자수/사업체수 기반이다. 따라서 서울 전체 읍면동의 최신 경제활동 집중도 반영에는 유용하지만, 산업별 구조까지 완전히 갱신한 것은 아니다.

## 후속 과제

1. 서울시 API의 산업분류별 행정동 자료를 더 세밀하게 파싱해 산업별 가중치 갱신을 시도한다.
2. 2015 산업별 가중치와 2024 전체 활동 가중치를 혼합하는 shrinkage 가중치를 검토한다.
3. 서울 외 시도는 동일한 최신 행정동 통계가 없으므로, 지자체별 자료 확보 가능성을 별도 조사한다.
