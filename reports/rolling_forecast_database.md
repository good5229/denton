# 롤링 외삽 데이터베이스

## 목적

기존 2023년 단일 외삽 검증을 확장해, 각 목표연도에 대해 이전 연도까지만 사용한 비례형 덴튼 외삽값을 데이터베이스 형태로 축적한다.

예를 들어 2022년 예측은 2015-2021년 연간 GRVA와 분기 지표만 사용하고, 2022년 분기 지표에 직전 `GVA / indicator` 비율을 적용해 생성한다. 실제 2022년 연간 GRVA가 있으면 예측 연간합과 비교한다.

## 산출물

| 파일 | 내용 |
| --- | --- |
| `data/processed/rolling_kosis_collected_all.csv` | 2015년 이후 장기 KOSIS 입력 데이터 |
| `data/processed/rolling_quarterly_gva_predictions.csv` | 목표연도별 분기 예측값 |
| `data/processed/rolling_annual_prediction_comparisons.csv` | 목표연도별 예측 연간합과 실제 연간 GRVA 비교 |
| `data/processed/rolling_prediction_summary.csv` | 전체 행 수와 평균 오차 요약 |

## 현재 수집 범위

- 연간 GRVA 요청 범위: 2015-2025
- 분기 지표 수집 범위: 2015Q1-2025Q4
- 비교 가능한 실제 연간 GRVA: 2017-2023
- 예측값만 보유하는 목표연도: 2024, 2025

## 실행 결과

- 분기 예측 행: 10,296
- 연간 비교 행: 2,574
- 실제값 비교 가능 행: 2,002
- 스킵 행: 12
- 2017-2023 평균 MAPE: 6.407741%

목표연도별 비교 가능 행은 매년 286개이며, 2024년과 2025년은 현재 연간 GRVA 실제값이 없어 예측값만 보유한다.

## 해석

이 구조를 쓰면 매년 새 연간 GRVA가 공개될 때 과거 예측값과 실제값을 비교할 수 있다. 또한 최신 분기 지표가 먼저 공개되는 기간에는 연간 GRVA 공개 전까지 2024년, 2025년과 같은 잠정 분기 GVA 예측 데이터베이스를 유지할 수 있다.

주의할 점은 2024년 이후 예측이 실제 연간 GRVA 제약으로 닫힌 값이 아니라, 마지막 벤치마크 분기의 비율을 유지한 외삽값이라는 점이다. 따라서 공개 전 모니터링 목적에는 적합하지만, 사후 확정 통계로 해석하면 안 된다.
