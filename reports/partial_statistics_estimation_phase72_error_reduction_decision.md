# 전산업 GVA 추정오차 축소 의사결정 기준

## 목적

현재 과제의 목표는 행정동·월·세부산업 값을 그럴듯하게 배분하는 데서 끝나지 않는다. 핵심은 **각 산업의 총부가가치(GVA)를 실제 상위 통계와 비교했을 때 얼마나 정확히 추정했는가**다. 따라서 이후 실험의 목적함수는 `실제 중분류 GVA - 추정 중분류 GVA`의 격차를 줄이는 것으로 둔다.

## 기본 판단

모든 산업을 개별 특화모델로 만드는 것은 비용 대비 효과가 낮다. 고양시와 포항시 모두 오차가 일부 산업군에 집중되어 있으므로, 이후 실험은 “전 산업 공통 기본식 → 취약 중분류 자동 선별 → 취약 유형별 활동지표 묶음 적용 → 실제 중분류·대분류 집계검증”의 4단 구조로 진행한다.

따라서 다음 라운드의 기본 전략은 **산업별 개별 수작업이 아니라 취약 원인군별 공통 처방**이다. 같은 상위산업 안에서 한 중분류는 과대, 다른 중분류는 과소인 경우에는 시 전체 총량 문제가 아니라 내부 분할 문제로 보고, 그 대분류 안에서만 금액을 재배분한다. 반대로 실제 GVA가 작아 상대오차가 큰 산업은 포스터의 핵심 성능 주장보다 자료보완 큐에 우선 배치한다.

## 이번 의사결정

이후 개선 목표는 `상위합계만 맞췄는가`가 아니라 **실제 중분류 총부가가치와 추정 중분류 총부가가치의 금액 격차를 줄였는가**로 고정한다. 포스터와 보고서에서도 합계 일치 자체를 성능으로 설명하지 않고, 산업별 표에는 `실제 억원`, `추정 억원`, `오차 억원`, `오차율`을 함께 제시한다.

모든 산업을 개별 특화하는 대신, 남은 취약 산업은 다음 7개 원인군으로 묶어 처리한다.

| 원인군 | 핵심 판단 | 우선 적용 자료 |
| --- | --- | --- |
| 생산시설형 | 사업체 수보다 설비 규모·대형 사업장 편중이 GVA를 좌우 | 제조업 부가가치·출하액, 전력, 공장면적, 산단 사업장 |
| 계약·공사형 | 종사자 수보다 착공·기성·계약 시점이 중요 | 건축허가·착공·사용승인, 공사계약액, 엔지니어링 수주 |
| 거래·자산형 | 거래금액·자산가치·임대면적이 생산활동에 가까움 | 공시가격, 거래금액, 연면적, 임대·관리 면적 |
| 이동·물량형 | 여객·화물·창고·항만 활동이 서로 다른 신호 | 승하차, 창고면적, 화물·항만 물동량 |
| 공공·비영리형 | 예산·시설정원·처리량이 일반 사업체 수와 불일치 | 보조금·위탁계약, 복지시설 정원, 환경처리량 |
| 디지털·콘텐츠형 | 매출·제작규모·플랫폼 활동이 소수 사업자에 집중 | 방송사업 매출, 콘텐츠 제작, 정보서비스·서버 규모 |
| 전문·지원서비스형 | 전문인력·임금·계약액이 단순 사업장 수보다 중요 | 고용보험 임금총액, 전문인력, R&D, 공공·민간 용역 |

실행 순서는 `취약 묶음 선정 → 복수 활동지표 후보 생성 → 상위산업 합계 보존 → 실제 중분류 GVA와 비교 → 오차금액 감소 후보만 채택`으로 둔다. 이 구조면 산업 74개를 하나씩 손보지 않고도, 오차가 발생하는 원인이 같은 산업군을 한 번에 줄일 수 있다.

## 오차 축소 운영 구조

| 구간 | 역할 | 적용 방식 | 채택 기준 |
| --- | --- | --- | --- |
| A. 공통 기본식 | 모든 산업의 기본 추정값 생성 | 사업체·종사자·지역 산업구조 등 반복갱신 가능한 공공자료 사용 | 전 산업에 동일 적용 가능 |
| B. 취약 산업 선별 | 성능 개선 대상 자동 추출 | 중분류별 실제 GVA와 추정 GVA를 비교해 금액오차·오차율 기준으로 큐 생성 | 오차율 50% 초과 또는 금액오차 1,000억원 초과 |
| C. 유형별 보강 | 비슷한 오차 원인을 가진 산업을 묶어 개선 | 제조시설면적, 사업체조사 매출·종사자, 건축허가, 물류·병상·복지시설 등 활동지표 적용 | 기존 추정보다 중분류 오차가 감소 |

## 선택 규칙

1. **금액오차 우선**: 상대오차가 커도 실제 GVA가 작은 산업은 후순위로 둔다.
2. **상위 산업 내부 분할 우선**: 같은 대분류 안에서 한 중분류는 과대, 다른 중분류는 과소라면 총량 문제가 아니라 내부 분할 문제로 본다.
3. **사업체 수 대체 기준**: 사업체 수가 생산활동을 설명하지 못하는 산업은 시설규모, 면적, 수송량, 병상, 거래금액, 계약액처럼 GVA와 더 가까운 활동량 지표를 사용한다.
4. **무료·반복갱신 가능성**: 공모전 산출물에 들어갈 모델은 무료 공공자료로 재현 가능해야 한다.
5. **성과 검증 방식**: 보강 전후를 모두 중분류 실제 GVA와 비교하고, 오차는 억원과 %를 함께 제시한다.

## 다음 격차 축소 원칙

현재 남은 문제는 “총량을 맞추는가”가 아니라 같은 상위산업 안에서 어떤 중분류에 GVA를 얼마나 배분할 것인가다. 그러므로 이후 개선은 아래 세 가지 규칙으로 제한한다.

| 규칙 | 의미 | 실행 방식 |
| --- | --- | --- |
| 원인군별 일괄 개선 | 모든 산업을 하나씩 특화하지 않고, 오차 원인이 비슷한 산업군을 함께 다룸 | 생산시설형, 계약·공사형, 거래·자산형, 이동·물량형, 공공·비영리형, 디지털·콘텐츠형으로 분리 |
| 단일 지표 대체 금지 | 공장 수, 매출, 종사자 중 하나만으로 갈아끼우면 일부 산업이 크게 악화됨 | 최소 2개 이상 활동지표를 조합하고, 상위산업 내부 합계가 유지되는 후보만 비교 |
| 개별 산업 악화 금지 | 전체 WAPE가 줄어도 특정 중분류 오차가 커지면 각 산업별 정확도 설명에 부적합 | 후보 채택 시 도시 전체 WAPE, 원인군 오차, 개별 중분류 오차를 동시에 검사 |

현재 결론은 이원화한다. 대외적으로 “각 산업별 정확도”를 강하게 말할 때는 개별 중분류 악화가 없는 엄격 기준을 사용한다. 반면 실제 격차를 더 줄이는 운영 단계에서는 판정 구간을 보호한 채 취약 중분류 오차를 줄이는 기준을 별도 후보로 관리한다. 두 기준을 섞지 않아야 전체 개선이 특정 산업 악화를 가리는 문제를 피할 수 있다.

## 포스터 반영 원칙

- 전면 성능지표는 `실제 GVA`, `추정 GVA`, `오차 억원`, `오차율`로 통일한다.
- 방어적 주석 대신 `실제 GVA`, `추정 GVA`, `오차 억원`, `오차율`을 병기한다.
- “프록시” 대신 `배분기준`, `활동지표`, `산업별 기준`처럼 심사자가 바로 이해할 수 있는 용어를 사용한다.
- 강점은 “전 산업을 동시에 추정하되, 오차가 큰 산업만 별도 기준으로 개선하는 재현 가능한 운영체계”로 제시한다.
