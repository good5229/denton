# A/B/D 산업 특화 Feature 수집

## 목적

농림어업(`A00`), 광업(`B00`), 전기·가스(`D00`)는 official actual 기준 상위 오차에 자주 등장한다. 이 산업들은 전체 산업 공통 모델보다 산업별 구조와 외생변수를 반영한 별도 모델 후보가 필요하다.

## 신규 수집 데이터

KOSIS `101 / DT_1K52F08`에서 시도×산업별 사업체수, 종사자수, 매출액을 수집했다.

| 항목 | 내용 |
| --- | --- |
| 기간 | 2020~2024 |
| 지역 | 전국 및 17개 시도 |
| 산업 | A00, B00, D00 |
| 지표 | 사업체수, 종사자수, 매출액 |
| 행 수 | 810 |

산출 파일은 `sector_structural_business_stats.csv`이며, MS Office 호환을 위해 CP949로 저장했다.

## 공표시점 정책

구조변수는 연간 조사자료라 target year 1월 1일에는 해당 target year 값이 공개되어 있을 수 없다. 모델링에서는 보수적으로 `target_year - 2` 이하 연도만 feature로 사용한다.

예를 들어 2024년을 예측할 때는 2022년 이하의 구조변수만 사용할 수 있다. 이 규칙은 데이터 유출을 막기 위한 최소 조건이다.

## 기존 활용 데이터

전기·가스는 기존에 수집한 FRED/ECOS 외생변수도 사용한다.

- WTI 유가
- 천연가스 가격
- 호주산 석탄 가격
- 원/달러 환율
- ECOS 수입물가지수 원유·유연탄·원유및천연가스
- ECOS 생산자물가지수 전력·가스·증기, 도시가스

분기 외생변수는 공표 지연 2개월을 가정하고, 예측 기준일 이전에 공표 가능한 분기만 사용한다.
