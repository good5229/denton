# 전기·가스 외생변수 보정 실험

## 목적

전기·가스 산업(`D00`)은 유가, 석탄 가격, 환율의 영향을 받을 가능성이 있다. 다만 상관관계가 있다는 사실만으로 기존 생산지수 기반 Denton 경로를 대체하면 과적합 위험이 크다. 이번 단계에서는 외생변수 보정이 실제 rolling backtest에서 개선되는지 별도로 검증했다.

## 실험 방식

기준선은 기존 `D00` rolling 연간 예측값이다. 타깃 연도별로 다음 절차를 적용했다.

1. 타깃 연도보다 과거의 `D00` backtest 행만 학습 자료로 사용한다.
2. 외생변수 후보별로 `actual / predicted` 보정비율을 단변량 회귀로 추정한다.
3. 학습 구간에서 기준선 MAPE보다 낮은 후보만 선택 대상으로 둔다.
4. 선택된 후보가 있으면 타깃 연도 외생변수로 보정계수를 계산한다.
5. 보정계수는 과도한 흔들림을 막기 위해 `0.75-1.25` 범위로 제한한다.

후보 변수는 연평균 수준값과 전년대비 변화율이다.

- WTI 유가
- 호주 석탄 가격
- USD/KRW 환율

천연가스 가격은 기존 수집 단계에서 FRED 시계열 수집에 실패했으므로 이번 실험에서 제외했다.

## 산출물

모든 CSV는 CP949로 저장했다.

| 파일 | 설명 |
| --- | --- |
| `energy_augmented_backtest.csv` | 지역별 D00 기준선 예측과 보정 예측 비교 |
| `energy_augmented_feature_diagnostics.csv` | 타깃 연도·후보 변수별 학습 구간 성능 |
| `energy_augmented_summary.csv` | 기준선 MAPE/WMAPE와 보정 MAPE/WMAPE 비교 |

## 현재 결과

실험은 2017-2023년 `D00` 비교 가능 행 126건을 대상으로 수행했다.

| 지표 | 기준선 | 외생변수 보정 |
| --- | ---: | ---: |
| MAPE | 16.103114% | 16.103114% |
| WMAPE | 9.813384% | 9.813384% |

`energy_augmented_summary.csv`의 `adopt_augmented_indicator`는 `no`다. 따라서 현재 단계에서는 외생변수를 실제 추정 경로에 채택하지 않고, 기존 생산지수 기반 Denton을 유지한다. 유가·석탄·환율은 설명 후보 또는 향후 데이터 확장 시 검토할 보조 지표로 남긴다.

## 해석

이 실험은 외생변수를 모든 산업에 무차별적으로 넣지 않고, 이론적으로 관련성이 높은 전기·가스 산업에만 제한적으로 적용했다. 또한 타깃 연도 이전 데이터만 학습에 사용했으므로 미래 정보 누수는 피했다. 다만 관측치가 많지 않고 외생변수는 전국 단일 계열이므로, 지역별 전력 사용량·발전원 믹스·요금 자료가 확보되기 전까지는 보조 진단으로 해석해야 한다.
