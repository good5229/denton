# 자료 공표시점 감사

## 결론

상위산업 actual 총량과 중분류 actual share는 기준연도 종료 다음날 바로 사용 가능한 자료가 아니다. 따라서 산출물은 두 갈래로 분리해야 한다.

- 속보성 지표: 전월·전분기 판단용. 해당 시점에 이미 공표된 월별 활동자료와 과거연도 구조만 사용.
- 정밀화 지표: 공표 후 재산출·검증용. 당해연도 GRDP, 사업체조사, 광업·제조업조사 등 공표 완료 자료 사용 가능.

## 확인한 공표시점 근거

| 자료 | 확인 공표/게시 시점 | 해석 |
| --- | --- | --- |
| 시군구 GRDP / 지역내총생산 | KOSIS Q&A에서 기준연도 익익년 12월 공표 인식 및 지자체별 개편 일정 차이를 확인 | 기준연도 직후 속보성 추정에는 사용할 수 없고, 공표 후 정밀화 기준값으로만 사용 |
| 포항시 2023년 기준 사업체조사 결과 | 포항시 통계정보 게시일 2025-01-07 | 2023년 중 또는 2024년 실시간 추정에는 사용 불가, 2025-01-07 이후 정밀화 가능 |
| 포항시 2023년 기준 광업·제조업조사 결과 | 포항시 통계정보 게시일 2025-02-28 | 제조업 정밀화 자료로만 사용 |
| 포항시 2024년 기준 사업체조사 결과 | 포항시 통계정보 게시일 2026-03-11 | 2024년 중 속보성 추정에는 사용 불가, 2026-03-11 이후 정밀화 가능 |

## 현재 산출물에 대한 판정

1. 포스터의 중분류 집계검증 표는 `2023년 연간 · 억원` 기준의 공표 후 정밀화 표다.
2. 정밀화 표의 실제값은 순수 중분류 GVA 공표값이 아니라 `대분류 GVA × 중분류 actual share` 환산값이다.
3. 해당 정밀화 오차는 예측 성능이 아니라 공표 후 구조 재산출·검증 성능이다.
4. 속보성 값은 월 변화·경보 후보 선별에만 사용해야 한다.

## 다음 운영 규칙

- 전월·전분기 속보 산출 시점에는 당해연도 GRDP, 당해연도 사업체조사, 당해연도 광업·제조업조사를 입력 금지한다.
- 공표 후 정밀화에서는 위 자료를 허용하되, 보고서와 포스터에 `공표 후 정밀화`라고 명시한다.
- 모든 원천자료는 `source_period`, `posted_at`, `retrieved_at`, `first_eligible_origin`, `allowed_track`을 갖는 vintage ledger에 등록한다.
