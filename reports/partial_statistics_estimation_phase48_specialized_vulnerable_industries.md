# 취약 산업 전용 배분모형 확장 검토

## 기준 문서에서 가져온 원칙

한국은행 이슈노트 「지역경기상황지수의 개발 및 활용」은 지역·산업별 고빈도 실제값이 없을 때 다음 구조를 쓴다.

1. 연간 지역소득처럼 상위 실제값이 있는 구간은 생산지표 움직임을 이용해 하위 시점으로 배분한다.
2. 아직 실제값이 없는 최근 구간은 직전 기준비율을 유지한다고 보고 외삽한다.
3. 건설업처럼 일반 생산지수가 맞지 않는 산업은 건설수주액을 누적 진행률로 기성액에 가깝게 변환한 뒤 별도로 추정한다.
4. 최종적으로 지역 합계·산업 합계·연간 합계가 상위 실제값과 맞는지 별도 검사한다.

이 원칙을 현재 프로젝트에 적용하면, 예측 취약 산업은 “전 산업 공통 사업체·종사자 배분기준”에 계속 묶어두기보다 산업별 전용 배분기준으로 분리해야 한다.

## 현재 취약 산업별 판정

| 산업 | 현재 문제 | 기존 오차 | 특화 가능성 | 1차 실험 방향 |
| --- | --- | ---: | --- | --- |
| 농림어업 | 생산액이 사업체·종사자보다 작목·경지·축산·위탁서비스 구조에 좌우 | 소분류 40.06%p | 높음 | A00 총량을 소분류 011·014로 배분하고, 직전 비중·경제총조사 매출·농업 물량자료를 비교 |
| 건설업 | 종합건설·전문공사의 계약금액과 원·하도급 구조가 고용과 불일치 | 중분류 31.60%p, 소분류 8.44%p | 높음 | 건설수주액을 건축 12분기·토목 24분기 진행액으로 변환해 건설업 전용 시간경로 작성 |
| 부동산업 | 임대·개발·중개서비스의 생산성이 사업체수와 정반대로 움직임 | 소분류 53.60%p | 높음, 단 자료 필요 | 건축물 연면적·공시가격·거래금액·임대료를 임대공급업, 중개업 인허가를 관련서비스업에 분리 적용 |
| 운수·창고업 | 여객·화물·창고·운송서비스가 서로 다른 물량 단위로 움직임 | 중분류 23.98%p, 소분류 13.02%p | 중간~높음, 단 자료 필요 | 화물차·여객·철도·항만·창고면적·택배 물동량을 소분류별로 분리 |

## 1. 농림어업 소분류 확장

가능하다. 다만 “소분류 실제 부가가치”가 있는 것은 아니므로, BOK 방식처럼 상위 A00 부가가치를 고정한 뒤 하위 소분류로 배분하고, 존재하는 매출·생산·면적 자료로 교차검증해야 한다.

현재 고양시 2015 경제총조사 기준 소분류 구조는 다음과 같다.

| 소분류 | 실제 매출비중 | 사업체·종사자 기준 예측비중 | 오차 |
| --- | ---: | ---: | ---: |
| 작물 재배업 | 4.70% | 44.76% | 40.06%p |
| 작물재배 및 축산 관련 서비스업 | 95.30% | 55.24% | 40.06%p |

따라서 농림어업은 중분류 A00을 시군구에만 잘 배분하는 데서 끝내면 안 된다. A00 내부에서 작물재배와 농업서비스를 분리해야 한다. 1차 모형은 다음 순서가 적절하다.

1. A00 연간·분기·월 총량은 상위합계로 고정한다.
2. 소분류 011·014 비중은 `직전 관측 비중 + 2015 경제총조사 매출비중`으로 초기화한다.
3. 무료 자료가 붙으면 011은 경지면적·작물생산액, 014는 농업서비스 사업체·위탁영농·축산지원 지표를 사용한다.
4. 검증은 소분류 합계가 A00과 일치하는지, 2015 매출 숨김검증에서 사업체·종사자 기준보다 나아지는지, 다른 시군구 표본에서 같은 방향인지로 판단한다.

## 2. 건설업 특화모형

가능성이 매우 높다. BOK 문서는 건설업을 일반 생산지수로 처리하지 않고 건설수주액을 기성액에 가깝게 변환한다. 현재 프로젝트에도 `construction_orders_by_region_type.csv`와 `rolling_construction_orders_by_region_type.csv`가 이미 있으며, 시도×분기×건축/토목 수주액을 포함한다.

적용 방식은 다음과 같다.

1. 건축 수주액은 향후 12분기에 나누어 진행액으로 반영한다.
2. 토목 수주액은 향후 24분기에 나누어 진행액으로 반영한다.
3. 시도별 건설업 분기 경로를 만든 뒤, 시군구·읍면동 공간배분은 건축허가·착공·사용승인 면적과 결합한다.
4. 검증은 중분류 41 종합건설업·42 전문직별 공사업, 소분류 411·412·421~426의 2015 매출 숨김검증과 상위 F00 합계 보존으로 한다.

단, 현재 수주 자료는 시도 단위다. 따라서 바로 개선 가능한 축은 “건설업의 시도·분기 시간경로”이고, 시군구·읍면동 공간 정확도를 개선하려면 건축물대장·건축허가·착공·사용승인 면적의 지역별 결합이 필요하다.

## 3. 부동산업 특화모형

필요성이 가장 크다. 고양시 소분류 기준으로 부동산 임대 및 공급업의 실제 매출비중은 60.56%인데 사업체·종사자 기준은 6.95%에 불과하다. 반대로 부동산 관련 서비스업은 실제 39.44%인데 사업체·종사자 기준은 93.05%로 과대 추정된다.

즉 부동산은 “사업체 수가 많은 중개·관리업”과 “부가가치가 큰 임대·공급업”을 분리해야 한다.

1. 부동산 임대 및 공급업: 건축물 연면적, 주택·상업용 건물 공시가격, 거래금액, 임대료 지수
2. 부동산 관련 서비스업: 부동산중개업 인허가, 공인중개사 사무소 수, 관리업체 수
3. 검증: 2015 경제총조사 매출 숨김검증에서 681/682 비중 개선 여부, 시군구별 부동산업 GVA 상위합계 보존 여부

현재 로컬에는 건축물대장 수집 흔적은 있지만, 부동산 거래·임대료·공시가격이 실험용 feature로 정리되어 있지는 않다. 따라서 부동산은 “즉시 실험”보다 “무료 자료 수집 후 1차 실험”이 맞다.

## 4. 운수·창고업 특화모형

가능하지만 단일 기준값으로는 안 된다. 운수·창고업은 철도, 여객, 화물, 소화물, 해상, 창고, 운송서비스가 서로 다른 물량 단위를 가진다.

현재 고양시 소분류 오차는 도로 화물 운송업 25.98%p, 기타 운송관련 서비스업 24.83%p, 육상 여객 운송업 19.58%p가 특히 크다.

적용 방식은 다음과 같다.

| 소분류 | 필요한 배분기준 |
| --- | --- |
| 철도 운송업 | 역 접근성, 승하차 인원 |
| 육상 여객 운송업 | 버스·택시 사업체, 정류장·노선·승객수 |
| 도로 화물 운송업 | 화물차 등록, 물류창고, 산단·공장 물동량 |
| 소화물 전문 운송업 | 택배 대리점·물류터미널·인구/사업체 배송수요 |
| 해상 운송업 | 항만 물동량, 어항·항만 접근성 |
| 보관 및 창고업 | 창고면적, 물류창고 등록, 냉동·일반 창고 구분 |
| 기타 운송관련 서비스업 | 터미널·항만·물류서비스 사업체와 물동량 결합 |

현재 `korea_transport_access_features.csv`는 구조는 있으나 거리값이 비어 있어 바로 성능개선 검증에 쓰기 어렵다. 포항은 항만·철강산단 특성 때문에 운수·창고 특화모형 효과가 고양보다 클 가능성이 높다.

## 다음 실험 우선순위

1. **농림어업 소분류 1차 실험**: 현재 자료만으로 바로 가능. A00 특화배분을 011·014로 낮추고 2015 매출 숨김검증으로 비교한다.
2. **건설업 시간경로 실험**: 현재 수주액 자료로 가능. 건축 12분기·토목 24분기 진행액 변환 후 F00 분기 배분을 기존 방식과 비교한다.
3. **부동산 자료 수집 후 실험**: 건축물 연면적·공시가격·거래금액이 필요하다.
4. **운수·창고 자료 수집 후 실험**: 항만·철도·화물차·창고면적·택배 자료가 필요하다.

## 포스터 표현 원칙

포스터에서는 내부 실험 용어를 쓰지 않는다.

| 내부 용어 | 포스터 표현 |
| --- | --- |
| 프록시 | 배분기준, 대체 기준값 |
| holdout | 숨김검증 |
| actual | 실제값 |
| MAE | 평균오차 |
| CV | 검증오차, 교차검증 |
| GVA | 부가가치 |
| 제약추정 | 상위합계 보존 추정 |

특히 `프록시`는 심사위원이 모형 구조를 모르면 의미가 전달되지 않는다. “사업체·종사자 기반 배분기준”, “농림어업 전용 배분기준”, “건설수주 기반 시간경로”처럼 데이터와 역할을 같이 적는 편이 낫다.
