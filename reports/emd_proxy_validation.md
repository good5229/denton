# 읍면동 Proxy 안정성 검증

## 목적

읍면동 단위 official GVA actual은 현재 확보되어 있지 않다. 따라서 읍면동 검증은 다음 두 가지로 나눈다.

1. 시군구 부모 GVA와 읍면동 합계가 맞는지 확인한다.
2. 서울시에서 확보한 2024 proxy를 사용해 2015 고정 proxy 배분의 안정성을 점검한다.

## 산출물

| 파일 | 설명 |
| --- | --- |
| `emd_proxy_stability_by_sector.csv` | 서울 읍면동 2015 proxy와 2024 proxy의 산업별 차이 |
| `emd_proxy_stability_by_sigungu_sector.csv` | 서울 자치구×산업별 proxy 안정성 |
| `emd_parent_constraint_summary.csv` | 전국 읍면동 배분의 시군구 부모 총량 정합성 |

## 해석

이 검증은 official GVA actual 검증이 아니다. 다만 2015년 고정 proxy가 2024년 proxy 구조와 얼마나 달라졌는지를 보여주므로, 읍면동 추정값의 노후화 위험을 평가할 수 있다. 차이가 큰 산업과 자치구는 최신 사업체 proxy 또는 상권/매출 자료를 우선적으로 보강해야 한다.

## 1차 결과

서울 읍면동 2015 proxy와 2024 proxy 비교는 산업별로 `425~428`개 읍면동 조합을 포함한다. 시군구 부모 총량 정합성은 모든 산업에서 통과했다.

proxy 구조 변화가 큰 산업은 다음과 같다.

| 산업 | absolute difference sum |
| --- | ---: |
| 금융 및 보험업(`K00`) | 50,900,403 |
| 정보통신업(`J00`) | 44,571,910 |
| 도매 및 소매업(`G00`) | 35,817,955 |
| 사업서비스업(`MN0`) | 35,524,321 |
| 부동산업(`L00`) | 26,330,664 |

이 결과는 읍면동 단위에서 2015년 고정 proxy를 장기간 사용하는 것이 특히 서비스 중심 업종에서 위험할 수 있음을 보여준다. 다음 단계에서는 서울처럼 최신 proxy가 있는 지역은 최신 proxy로 갱신하고, 전국 확장에는 지역별 사업체/매출 proxy 갱신 주기를 추적하는 방식이 필요하다.
